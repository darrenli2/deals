## Requirement
* ruby `2.3.4`
* yarn `1.17.3`

## How to run the application

* `bundle install`
* `yarn install`
* `bundle exec rails s`

## How to run specs
* `bundle exec rspec`

## Notes
* For security reason master.key should not be checked in to repo. It is checked in just for convenience purpose.
